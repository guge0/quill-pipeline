# T2 清单与等待

剧情要点:
1. 周五下午,苏蔓如约把苏皓的通话清单送来,整整三十七页。两人在江叙白住处楼下的茶餐厅对坐,本章以整理线索与人物情绪为主,事件密度低。
2. 清单里最后一通电话时长两分十四秒,号码归属显示为"宏远物流"。苏蔓说弟弟从没提过任何物流公司。
3. 苏蔓回忆弟弟失踪当晚穿一件灰色连帽衫出门,说要"去取个东西,很快回来"。
4. 雨又下起来。江叙白左肩旧伤隐隐作痛,他没说,只把清单第三十七页折了角。
5. 章末两人沉默告别,江叙白决定档案馆之约后再去碰宏远物流。

章末钩子:茶餐厅玻璃外,一把黑伞在雨里停了很久,伞下的人始终没进来。
