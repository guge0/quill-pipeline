# T3 档案馆碰头

剧情要点:
1. 周日上午十点,江叙白准时到市档案馆三楼。何沛已在,手里捏着调阅函——他守了约,江叙白这三天也守了约,没单独进当铺。
2. 管理员老覃验过调阅函,从库房取出档案编号A-113的卷宗:1998年"江湾码头沉箱案"。
3. 三人翻卷时发现卷宗缺了第三页,借阅登记卡上最后一次调阅记录的签名栏被水渍洇开,辨认不出。
4. 何沛脸色变了——以他刑侦支队副队长的权限,这卷宗本不该有人能在登记不全的情况下调走内页。
5. 章末,老覃迟疑地说了一句:上一个来调这卷的人,也是个雨天来的。

章末钩子:江叙白口袋里,那把07号黄铜钥匙硌着掌心。
