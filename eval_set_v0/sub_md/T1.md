# T1 夜访回声巷

剧情要点:
1. 周四夜里下起雨。江叙白十一点二十抵达回声巷巷口,在报刊亭老板的闲谈里确认传言:整条巷子只有回声巷17号在雨夜亮灯——那家旧货当铺守拙斋。
2. 江叙白以避雨为名进店。聂守仁端坐柜台后,左手戴着黑色手套,自始至终没摘。店堂尽头的卷帘门内侧还有第二道门,挂着锁。
3. 聂守仁话里有话,送客前却把一把黄铜钥匙放在柜台上,钥匙柄刻着编号07,说"先生若有旧物要赎,凭这个来"。江叙白没接,聂守仁把钥匙塞进了他口袋。
4. 出店后江叙白在巷尾调取自装的微型记录仪,发现守拙斋门口路段的录像缺失了四十一分钟——恰好覆盖他在店内的时间。
5. 凌晨,江叙白与市局刑侦支队副队长何沛在江湾区一家粥铺碰面。何沛透露:三个月里六名失踪者的卷宗他翻过,有四起结案材料"干净得过分"。
6. 两人约定:三天后上午十点在市档案馆三楼碰头,何沛带调阅函来,调档案编号A-113的旧卷;在那之前,江叙白不单独进当铺。
7. 临别何沛补一句:苏蔓那边周五下午会把苏皓的通话清单送到江叙白手上,清单是她跑了三趟营业厅才开出来的。

章末钩子:江叙白回到住处,掏出那把07号黄铜钥匙——钥匙柄内侧还刻着第二行小字,他凑到灯下,看清的瞬间握紧了钥匙。(小字内容本章不揭示)
