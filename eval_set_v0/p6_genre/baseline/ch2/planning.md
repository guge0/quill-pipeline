# 第 2 章 创作者细纲

## 戏核
这一章的存在是为了让**信息差在无声中拉紧**。读者读完这章，记住的东西应该是**苏蔓递来的清单里藏着一条线，而窗外有人正在听**。

## 戏核如何承载（含技法）

本章的核心判断是：**事件密度低，但心理密度必须高。** 这不是推进情节的章节，而是**拉紧弦**的章节——通过两个人围坐于一盏灯下的对话，让上一章获得的冷硬线索（07号钥匙、守拙斋）变得更具体、更有人味，同时埋入下一枚“被注视”的钉子。

承载方式如下：

**第一层：清单作为“静物特写”。** 三十七页通话清单，本身是枯燥的数据。但要用江叙白的翻页动作和视线停留来赋予它节奏——当他翻到第103号通话（结尾的通话记录，共37页，第103号），那两分十四秒、宏远物流，镜头必须**慢下来**，仿佛电影里主角的手指停在某个数字上不动。让读者跟着他的目光一起看到那个号码、那个归属地。苏蔓的插话“他没提过任何物流公司”是对这个细节的第一次加注——不是信息，而是情绪：弟弟有秘密。

**第二层：苏蔓的回忆是长镜头下的焦虑特写。** 她回忆苏皓出门时穿灰色连帽衫，说“去取个东西，很快回来”。这里要用**近景跟拍**：她低着头，手指无意识地拨弄杯沿，声音平稳但每次提到细节时会顿一下。这不仅是在交代线索，更是在**让读者进入她的视线**——她记得每一个细节因为她在脑子里重放了无数次。她的克制与江叙白的克制形成一次无声的对位：两个都在憋着情绪的人，对着三十七页纸，谁也不说“我害怕”。

**第三层：肩膀的痛与窗外的人。** 雨又下起来，江叙白左肩旧伤隐隐作痛，但他没说。这个身体的“失语”与章末那把黑伞的“驻足”构成**一次对照**：身体在替他说话（痛是他对危险的记忆编码），而窗外的伞在替某种威胁发出沉默的宣告。这里要用**蒙太奇式的切镜头**：先拍江叙白微微蹙眉、右手不自觉地按了下左肩（动作要短、要收），然后切到窗外雨幕里那把黑伞——静止的、朝向他们的、撑了很久。章末，伞终于移出画面，但读者会一直记得它在。

**第四层：沉默是这章的呼吸。** 两个人说完该说的话，没有多余的寒暄。苏蔓起身离开时江叙白没送，他在想“档案馆之约后再去碰宏远物流”——这个内心决定要写得干净，不铺垫、不解释。他折起清单第三十七页角这个动作，是对所有未解之事的**物理标记**。

## 笔墨分配

- **关键处**（中段至章末，约占70%）：清单发现宏远物流的段落、苏蔓回忆弟弟出门的段落、章末黑伞段落。这三个场景是这章的戏核所在——第一个拉开线索，第二个注入情感重量，第三个埋入威胁信号。每个都要展开，要有细微动作和停顿。

- **过渡处**（开篇，约占15%）：两人在茶餐厅落座、苏蔓从包里取出清单的过程。写快速、写现实——点单、倒水、低头翻包，不必展开氛围描写。

- **删减判断**（约占15%）：雨本身不要单独写一场或用大段环境描写来渲染。雨的作用是“让左肩疼”和“让伞存在”，仅此两项功能。不展开雨的意象，不写景不抒情。另外，江叙白的调查计划（档案馆之约后再去宏远物流）只在章末一句带过即可，不必展开推理过程。

## 与前后章的关系

- **承**：第1章留下的守拙斋谜团、07号钥匙、聂守仁的警惕性，在这一章被清单里的“宏远物流”接上——这是第一条从已知线索里长出来的新线索。同时苏蔓的出场让上一章的“委托人”形象从纸面变成有血有肉的人。

- **启**：宏远物流成为下一章的行动目标；窗外黑伞的存在暗示有人正在监视这场见面，下一章可能会引出跟踪线或新的第三方角色——这条线不急着揭示伞下是谁，但要让读者知道“他们被看见了”。
