# 第3章 创作者细纲

## 戏核
这一章的存在是为了 **让“A-113档案”从线索编号变成具象的威胁——让读者亲眼看到档案被人动过，感受到对手不仅存在，而且权限比何沛还高。**
读者读完这章，记住的东西应该是 **“第三页被撕掉了，而且有人能在登记不全的情况下调走它。”** 不是档案内容，是档案被干预的事实本身。

## 戏核如何承载（含技法）

核心场景在 **中后段——翻卷、发现缺页、查登记卡、何沛脸色变化**。这里要用 **慢节奏+特写** 来放大颗粒度：老覃离开后三人各自坐下，江叙白的手指掀开封面的触感，纸张的脆度，编号钢印的压痕。翻到缺页处时不马上揭示——先让江叙白翻完前三页，发现页码从1跳到2、2跳到4，他停顿，往回翻确认，再翻到装订线残留的纸茬。这个“确认的过程”本身就是情绪积累，比直接写“缺了第三页”有力。

何沛接过卷宗后，他的反应是戏核的第二层冲击。此处用 **职业反应的细微描写** 来制造信息差：他是专业刑警，他能从装订痕迹看出是近期被撕的，还是原件归档时就缺的。他手指摸过纸茬时指腹的停顿，是他读懂了什么。他翻到登记卡时沉默的那几秒——读者不知道他在想什么，只知道他脸色变了。之后他开口说“以我的权限，这本不该发生”时，语气里不应该有愤怒，而应该是 **冷静的、审视的、带着职业警觉的确认**。这种“连副队长都觉得不对劲”本身，比直接写“有人在背后操控”更让读者不寒而栗。

章末老覃的那句话要用 **轻处理**。不要给悬念式的停顿或加重语气，就让他像说一句日常提醒一样随口带出来：“上一个来调这卷的人，也是个雨天来的。”老覃说话时正在收拾抽屉，背影对着他们，语气平淡。这种“说话的人自己没意识到信息量”的处理，比刻意渲染更有效。结尾切到江叙白口袋里铜钥匙硌掌心的触感——用一个 **触觉特写** 收束全章，把“档案被干预”和“钥匙/守拙斋”两个线索在感官层面并置。

## 笔墨分配

- **关键处（中后段，约占全章50%）**：翻卷宗发现缺页 → 何沛验登记卡 → 两人无声的信息交换。这一段要展开，让每一个动作都有重量。何沛脸色变化的瞬间要写细，写他职业身份的裂缝——一个老刑警需要多少证据才会在别人面前露出破绽，这里就是他破绽出现的时刻。
  
- **过渡处（开篇至中段前，约占30%）**：江叙白抵达档案馆、与何沛碰面寒暄、老覃验证调阅函、取卷过程。这些段落要写得快、干净，只保留必要的场景说明书（环境、动作、简短的对话），不展开心理描写或氛围渲染。特别是何沛守约和江叙白守约这个信息，一句带过即可，它只服务于“两人之间信任关系的确认”。

- **删减判断**：sub-md中“何沛脸色变了——以他刑侦支队副队长的权限，这卷宗本不该有人能在登记不全的情况下调走内页”这句话背后的心理活动，正文中 **不展开写**。何沛作为刑侦副队长，他的警觉应该通过动作和微表情表现，而不是通过内心独白。读者自己能从“他不说话了”“他翻了两遍登记卡”这些细节里读懂他的态度。另外，“这三天江叙白没单独进当铺”这个约定执行情况，正文中只需要何沛问一句、江叙白回答一句，不做展开或回忆。

## 与前后章的关系

- **承（接Ch02）**：Ch02结尾，江叙白从守拙斋拿到钥匙、意识到守拙斋已物理消失、发现黑伞神秘人。本章开篇就要接住 **何沛约法三章后的执行情况**（两人都守约了），以及 **江叙白口袋里那把07号钥匙**——这是物理锚点，也是情绪锚点。钥匙在本章不展开解释，但要让读者感觉到它在江叙白身上、在提醒着他什么。

- **启（给Ch04）**：本章留下的核心任务是线索导向——**“谁调走了第三页”**。这将驱动Ch04的调查方向：何沛会用自己的渠道去查那本登记卡的时间线、查值班表、查谁有权在非登记状态下调卷。同时，老覃那句“雨天”会让江叙白把“档案干预”与“守拙斋只在雨夜亮灯”这两个线索做连接——Ch04可以开始让这两条线产生交叉。

## 硬信息锚点块

- **时间**: 周日上午十点；三天（何沛的约定周期）
- **地点**: 市档案馆三楼
- **人物**: 江叙白；何沛（刑侦支队副队长）；老覃（管理员）
- **数字**: A-113（档案编号）；1998年（沉箱案年份）；第三页（缺失页码）
- **约定**: 何沛守约提供调阅函；江叙白守约三天内未单独进当铺
- **设定**: 调阅函（正式手续）；借阅登记卡（有签名栏）；登记卡上最后一次签名被水渍洇开；档案缺第三页
