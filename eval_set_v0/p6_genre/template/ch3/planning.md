# 第3章 创作者细纲

## 戏核
这一章的存在是为了**把线索从“散点收集”推进到“有机关联”**——让读者明确意识到：苏皓案、守拙斋、1998年沉箱案、黄铜钥匙，这四条线不是孤立的，它们被同一只手翻动过。

读者读完这章，记住的东西应该是**“有人比我们更早动过这卷宗，而且何沛的权限都挡不住”**——不是缺页本身，是缺页背后那个“能够绕过程序”的人。

## 戏核如何承载（含技法）

核心场景是**三个人围着卷宗桌，发现缺页的那一瞬间**。这个场景要慢、要静、要写实——不是动作戏，是看不见的压力的积聚。

开篇直接进档案馆三楼环境，用长镜头推进：江叙白上楼，看到何沛已经在等。二人点头，没有寒暄（上一章末尾的朗逸和黑伞已经制造了足够的紧张，这里不需要再铺垫气氛）。老覃出场，验函，取卷——这个过程的节奏要均匀，像走程序一样平淡，目的是制造“一切正常”的假象，为后面的突变蓄力。

翻卷宗的过程用几组特写切换：江叙白的手指划过纸张边缘、老覃的钢笔指着页码、何沛微皱的眉头。三人的视线在第三页缺失处的断口停留——这里**必须给一个静帧式的停顿**，让沉默占据两三秒，然后才接对话。这个空白是情绪的放大器，比任何台词都有效。

借阅登记卡的发现是第二层推力：签名栏被水渍洇开。何沛的警觉反应才是真正的重笔——他作为副队长，太清楚这个信息意味着什么：要么是内部人违规操作，要么是有人掌握了超越权限的方法。这里要写他的微表情变化，写他下意识去摸烟又停下的小动作（室内禁烟），让读者感知到他的不安大于愤怒。

老覃的收尾台词是关键的一个反转：“上一个来调这卷的人，也是个雨天来的。”——这句话要轻、要迟滞，像随口提起，但落在氛围里就是一块石头。这句话把“雨夜”这个符号从守拙斋延伸到了档案系统，暗示那条雨夜线索不是孤例。

章末钩子用两个短句解决：江叙白口袋里，钥匙硌着掌心。这个触觉细节要接在“雨天”一词之后——雨天呼应他的旧伤隐痛（设定），钥匙的质感呼应“1998”。用身体感受代替心理描写，“硌”字就够了。

## 笔墨分配

- **关键处**（约占全章50%）：档案室里三人翻卷宗的过程，从“一切正常”到“发现缺页”的转折，借阅登记卡的发现，何沛的反应。中段到章末之间的连续段落，慢节奏展开。
- **过渡处**（约占30%）：江叙白出门去档案馆的简要过程、进入大楼、与何沛简短碰面。这些段落一笔带过即可，作用是承上启下，不抢戏。
- **删减判断**：档案馆环境的全景描写可以不展开，只写“三楼”“旧铁皮柜”“日光灯管的嗡鸣”三个意象就够了。老覃的长相、动作细节点到为止，不是本场重点。三人之间的寒暄（吃饭了吗、天气如何）全部删掉——这个场景不需要社交润滑，直接进正题。

## 与前后章的关系

- **承**：Ch2结尾的黑伞人和朗逸带来的压迫延续到这里——江叙白和何沛的合作是“在压力下进行”的设定，不需要明说，但读者能从二人简洁的对话中感知到上一章的余波。通话记录中的宏远物流线索放在这一章的重叠处被再次提及（江叙白在翻卷宗间隙扫一眼档案编号A-113的年份，与钥匙上1998形成对照），完成线索的第一次交叉验证。
- **启**：这一章留下的缺口（第三页内容、绕过程序的调阅者、老覃的“雨夜”）为Ch4提供具体调查方向——江叙白会去追查谁在何沛之前调阅过卷宗、为什么选雨夜、那页纸上写了什么。同时，“雨天”这个词第一次把“雨夜”的符号正式锚定到调查线上，为后续所有雨夜场景建立暗示体系。

## 硬信息锚点块

- **时间**：周日上午十点；这三天（约定守约的时间）；雨天（老覃提到上一次调阅也是雨天）
- **地点**：市档案馆三楼；库房；江湾码头（卷宗案发地点，通过翻卷宗时视线扫过文件标题间接提及）
- **人物**：江叙白；何沛（刑侦支队副队长）；老覃（档案馆三楼管理员）
- **数字**：A-113（档案编号）；1998（沉箱案年份，钥匙上刻印）；第三页（缺失页码）
- **约定**：何沛守了约（拿到调阅函）；江叙白守了约（三天内没单独进当铺）；调阅函（由何沛提供）
- **设定**：市档案馆位于江湾区，旧案卷宗在三楼；借阅登记卡的签名栏被水渍洇开（本场景新增设定）；老覃“认调阅函不认人”的性格（本章通过动作体现）
