# 第1章 创作者细纲

## 戏核
这一章的存在是为了**同时建立三条互不交叉但指向同一核心的信息链——江叙白的物理调查线、录像的证据线、何沛的体制内线索线，使读者在章末意识到“三个方向各自发现异常，但异常的形状不同”，从而诱发对“这些碎片到底拼成什么”的追问**。
读者读完这章，记住的东西应该是**“守拙斋里的时空是断裂的——41分钟凭空消失，而何沛那边也有一模一样的‘干净得过分’——整个事件不是表面上的失踪案”**。

## 戏核如何承载（含技法）

本章的核心技法选择是**“节奏错位与信息层递”**。前半段的夜访场景采用**长镜头式跟随**——从江叙白进巷到出店，镜头几乎不切，全程押在他的视觉和听觉上。报刊亭老板的闲谈做“铺垫”，进店后聂守仁的每一句话都落点精准、句句有刀。这里用**慢节奏对话**制造压迫感：聂守仁不说一句多余的寒暄，几句话就把“我知道你是谁”“我知道你为什么来”“但我不接你的话”三个信息压进台词里。这一段的核心抓手是**那把黄铜钥匙——它同时是物理道具、叙事符号和信息门槛**。

中段（出店后巷尾查看录像）是本章的**情绪反转瞬间**。前面所有的安静、克制、看似平淡的对话在这里被骤然打破：录像缺失41分钟。技法上采用**冷刀切入**——不渲染、不配情绪，直接呈现事实。江叙白确认录像抹除的过程用**特写+静默**：手指翻动设备菜单的画面拉长，他再次抬头看深巷的镜头切回慢速，读者和主角同时感知到“刚才那个平静的十分钟，在现实中是空白的”。

章末粥铺场景换**快切对话**。何沛的每一句话都是单刀直入的信息输出——结案材料“干净”、四起异常、六名失踪者——节奏紧凑，与前段的慢形成对比。这里不做情绪渲染，因为信息本身就是情绪。末段江叙白回住处看钥匙，用**降速特写**收尾：钥匙柄转过来，第二行小字出现，握紧，黑屏。这一瞬间的作用不是揭示，而是**封存悬念**——读者知道有了新信息，但不知道是什么，这正是翻页的动力。

## 笔墨分配

- **关键处（约占全章 50%）**：守拙斋进店场景（聂守仁的对话、钥匙递出）与巷尾录像缺失的揭示。这两段是戏核所在，要细、要稳。前半段压节奏营造压迫感，后半段用冷刀切入制造冲击。江叙白看录像缺失的反应镜头必须留白，不给心理描写，让读者自己填。
- **过渡处（约占全章 35%）**：开场报刊亭对话、粥铺信息交换。报刊亭对话只服务于“确认传说”和“给读者一个对守拙斋的预期”，不展开人物关系。粥铺场景信息密度高但叙事位置低——它是在落实承诺、设定期限、铺垫下一步，本身不是高潮，因此笔墨简练，用快切完成，不给环境描写和心理描写。
- **删减判断（约占全章 15%）**：苏蔓通话清单的提到只需一笔带过，目前阶段不需要展开苏蔓如何拿到清单的过程。江叙白从巷口走到巷尾的沿途描写全部删减，只靠声音和灯光的推拉来暗示空间推进。章末钥匙小字只写“握紧”不写内容，不写江叙白此刻的心理活动。

## 与前后章的关系

- **承**：承接前文苏皓失踪案以“基站定位回声巷”收尾（已获线索c001），本章以“夜访回声巷”作为对这条线索的首次实体落地。何沛出场承接前文“请市局旧同事帮忙”的未写约定，钩子001、002（苏皓最后通话对象、警方快速结案原因）在本章半开——何沛暗示了异常但没有答案。
- **启**：本章留下的三个前置条件直接驱动下一章：①录像缺失的真实原因（设备故障/人为抹除/超自然？）→ ②A-113号卷宗的存在（三天后开馆调阅）→ ③钥匙上的第二行小字（章末钩子，下一章头揭示）。这三个钩子各自指向不同方向，但都锚定在“守拙斋和聂守仁是核心”这一共识上，下一章即使没揭底，读者也知道故事在朝深水区推进。
