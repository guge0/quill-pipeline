何沛的黑色桑塔纳停在市档案馆台阶下，引擎已经熄了。他摇下半截车窗，左手搭在窗沿上，指间夹着一根没点的烟。看到江叙白从街角拐过来，他把烟塞回烟盒。

“老覃八点五十就开好门了。”何沛推开车门，没寒暄，“我让他先别上三楼，在二楼报刊室等着。”

江叙白看了眼手机上的时间。九点四十三。他今天提前出门，在隆兴路绕了一圈。苏皓照片里那座老宅的楼下，现在是个快递代收点，卷帘门半拉着，门口堆了十来个纸箱，全是宏远物流的封箱胶带。他没拍照片，只是站在对面早点铺子里买了个包子，看那些纸箱看了十分钟。

“周六上午的档案馆，除了我们没人来。”何沛锁好车，把调阅函折了两折插进外套内侧口袋，“待会儿上去之后，老覃验函需要点时间，你别催他。”

江叙白没回话。左肩从凌晨开始隐隐发酸，现在酸感已经蔓延到肩胛骨内侧，像一根细钢丝嵌在骨缝里被慢慢拧紧。天气预报说今明两天多云，但肩膀从来不信天气预报。

档案馆大厅的地面刚拖过，水渍还没完全干透，空气里混着清洁剂和旧纸张的气味。老覃站在二楼楼梯口，手里拎着一串钥匙，钥匙环上挂了至少二十把。他看见何沛身后跟着的人，目光在江叙白脸上停了不到一秒，然后移开。

“覃老师，这位是我朋友，调阅函上写了。”何沛把调阅函递过去。

老覃接过来，没急着看。他把钥匙串换到左手，右手从胸前的口袋里摸出一副老花镜，镜腿边缘的镀层已经磨出了铜色底。他戴上眼镜，把调阅函举到眼前，拇指着内容一行行往下指，嘴唇微动，像在默念。念完了，他把调阅函翻过来看了背面，又翻回来，盯着市局的公章看了足有十秒钟。

“三楼。”他把调阅函还给何沛，“A字号段的卷宗在东北角铁皮柜，第三排。你们先在阅览室等我，我去取。”

“不用登记本签字？”何沛问。

“调阅函留在我这儿。”老覃已经开始往三楼走，脚步不快，但每一步都踩在楼梯中间那道被磨得发亮的木纹上，“等人走了再签。”

阅览室在三楼走廊的尽头，是个长方形房间，朝南的窗户占了半面墙，窗外是档案馆后院几棵老樟树。窗玻璃上有层薄灰，天光透进来被滤成淡黄色。屋里摆着四张木桌，每张桌子配两把椅子，桌面铺着墨绿色软胶垫，胶垫边缘已经老化发黄。

何沛拉了把椅子坐下，把烟盒掏出来放在桌上，没点。江叙白站在窗前往外看了一眼，楼下后院里停着一辆报废的档案转运车，轮胎瘪了，车厢上的白漆字“临江市档案馆”掉了半边。

走廊里传来钥匙碰撞的声音，由远及近。老覃端着一个浅灰色档案盒走进来，盒子是硬纸板做的，四角包着深蓝色布边，盒盖上贴了张白色标签，上面用钢笔写着“A-113 江湾码头沉箱案 一九九八年”。

他把盒子放在桌上，没打开。从裤兜里摸出一副白棉手套，放在盒盖旁边。

“田七。”他说。

何沛愣了一下。

“你们要看的卷宗，”老覃解释道，“翻的时候戴手套。里面的纸已经二十多年了。”

他说完在江叙白对面的椅子上坐下，但椅子离桌子稍远，后背不靠椅背，双手搁在大腿上。他会留下来。江叙白从他的坐姿里读出了这个信息。

江叙白戴上手套，解开档案盒侧面系着的纸绳。那是普通的档案绑绳，细棉纸搓成的，打了个活结，绳头留了两寸长，末端剪成燕尾口。绳子已经泛黄，但结扣处没积灰尘。

盒盖打开的瞬间，一股干燥的旧纸味散出来。里面整整齐齐叠放着八开大小的卷宗纸，纸质发黄但没脆，边缘有几处水渍印子，但文字部分清晰。江叙白把整沓卷宗托出来放在桌面上，分量比看上去的轻。

第一页是案件基本情况表。他用指尖从页首往下滑，停住。

案件编号：A-113。案件名称：江湾码头沉箱案。立案时间：一九九八年九月十八日。承办单位：临江市公安局刑侦支队。案件性质：推定杀人。被害人：李兆龙，四十七岁，江湾码头十二号仓库库管员。尸体未寻获。

何沛探过身来看，呼吸声在安静的阅览室里显得略重。江叙白可以把卷宗推给他看，但他没推。何沛需要从自己的角度读，读他作为刑警应该发现的东西。

江叙白翻到第二页。现场勘验记录。沉箱位于十二号仓库后方水域，距岸九米，水深四米三。沉箱尺寸一米二乘一米二，内壁有喷溅状血迹，血型A型，经比对与李兆龙社保卡归档血型一致。箱盖从外部被焊接封死，焊点为六处，间隔均匀，手法专业。箱体表面除中心区一处手印外，无其他指纹痕迹。

翻页。

第三页应该续接第二页的勘验记录后半段，但江叙白拇指捻过去的时候，纸页的重量突然不对了。

他低头看装订口。

第三页被撕掉了。

不是裁纸刀裁的，不是剪刀剪的。纸根还残留在装订处，撕口是锯齿状的，上下宽中间窄，像是一只手捏住页脚斜着撕扯下来的。撕口处的纸纤维比周围更黄，意味着这页缺失已经很久了。至少在近几次有人翻阅之前，它就不在了。

何沛注意到了他的停顿。他从对面伸出手，江叙白把缺页的位置让给他看。

何沛没说话。他把整沓卷宗轻轻拿过去，将装订口举到眼睛的高度，对着窗户方向看。那个姿势保持了大概五秒，然后他放下卷宗，翻到最后面，用拇指拨开订书针的背面。三枚，没拆过，针脚是原来的一次性装订。

“档案编号和页码检查是你来之前我刚要求的流程。”何沛说，声音压得不高，但语速慢了半拍，“去年省厅下发新规，所有存档案卷宗在借出归还时必须核验页数，登记缺页情况。一九九八年这批借阅记录应该从归档日起就有完整的登记卡。”

他把卷宗盒转过来，抽出插在盒盖内侧纸槽里的一张硬质登记卡。卡片是标准印制的，竖格横栏，从九八年至今的借阅记录逐行填写：日期、借阅人、证件号、归还日期、核验人。卡片上总共只有四条记录，都在九八年九月底到十月初之间，借阅人全是当年办案组的成员，最末一行是“一九九八年十月十七日 归档”。

核验人那一栏，四次借阅全部没填。

不止这个。归还日期里有两次写的是“当日”，一次写的是“次日”，最后一次借阅时间写的是“十月十五日”，归还日期却是空白。

何沛的拇指按住那条空白的归还栏，指节发白。江叙白认识这个动作。何沛在刑侦支队时，翻到口供前后矛盾的地方就是这个反应，不是愤怒，是脑子已经在同时跑三条推断线。

“这卷一九九八年归档后，理论上没人再调过。”何沛把登记卡平铺在桌上，“但缺了第三页。装订口是撕扯方式取下，不是归档后损坏。”

他顿了顿。

“这个卷宗，以我的权限，不该有人在登记不全的情况下调走内页。”

何沛没说“有人动了档案”，他说的是“不该有人能”。江叙白听到了那个“能”字，意味着何沛在怀疑的不是某个具体的人，而是程序上的漏洞意味着什么。要么是比何沛权限更高的人绕过了登记系统，要么是管理层内部有系统性漏洞。

一直在旁边沉默的老覃忽然动了。他从椅子上站起来，走到桌前，从褂子口袋里取出一块绒布，开始擦他的老花镜片。一圈一圈，慢悠悠地擦，沿逆时针方向。

“你们还在看。”他说，声音不大，像是对着桌子说的。

江叙白取下右手的棉手套，从夹克内侧口袋里摸出手机，把缺页的装订口和登记卡分别拍下来。何沛看他拍完，把卷宗合回去，纸绳重新系好，但这次绳结打在了盒盖正面而不是侧面。一个刑侦人员的职业习惯：处理过的物证要留标记，方便后来者知道有人动过。

“覃老师。”江叙白站起来，走到老覃面前，“这几年有人来调过A-113吗？”

老覃还在擦镜片。他擦镜片的方式很特别，不是擦完就戴上，而是擦干净之后举到眼前，对着光看镜片上还有没有指纹印，看完了继续擦。

“登记卡上的记录断了之后，有人来问过吗？”江叙白又问了一遍。

老覃放下绒布。他把眼镜慢慢戴上，两只手扶着镜腿，调整了两次位置。然后他抬起眼睛，隔着镜片看向江叙白。

“上一个来调这卷的人，”他说，“也是个雨天来的。”

阅览室的窗玻璃上忽然滑下一道水痕。雨不知什么时候下起来的，细密无声，直到蓄够了分量才顺着玻璃流下来。院子里樟树的叶子开始滴水，打在报废转运车的铁皮车厢上，声音闷而短促。

江叙白看着老覃。老覃说完这句话之后就没再开口，他伸手把档案盒往桌子中央推了推，推到正对着吊灯的位置，然后退后一步，像是在说“你们可以走了”。

何沛把烟盒收进口袋，站起来拍了拍裤子。他在楼梯间等江叙白，两个往下走的时候，谁都没说话。一直走到二楼拐角，何沛停住了。

“你反应过来了吗。”他说，不是问句的语调。

江叙白当然反应过来了。老覃说的是“也是个雨天来的”——“也”。今天没下雨，直到几分钟前才落的雨。但他记得老覃用的词，是那个“也”字。

他在把第四个失踪者档案放进车后备箱的那个下午，天也是突然下雨的。他在茶餐厅和苏蔓见面的晚上，路灯下黑伞人的伞面上水花溅白。他去守拙斋的那个夜里，聂守仁说“下雨天，才有人想起当掉点什么”。现在老覃告诉他，上一个调A-113卷宗的人来的时候，也下雨。

巧合出现一次是巧合。出现三次是安排。这是第四次。

“这周日下午我还有半天假，”何沛在楼梯上侧过身看着他，“你要查上一个调卷人的身份。我可以帮你调档案馆大厅的监控。但一楼入口那台摄像头的硬盘只保留十五天，如果老覃说的‘上一个’超过这个时限，你得给我别的数据源。”

“先确认十五天内的。”江叙白说，“查不到再往前推。”

何沛点了下头。他没问“你打算怎么查”，也没问“查出来之后要做什么”。五年前他们搭档的时候，何沛从来不问这些问题。他只是每次在收工前说一句“别让我后悔”。

走出档案馆大门时，雨已经下紧了。台阶下的积水里漂着几片樟树叶子，何沛的黑色桑塔纳雨刷器夹了张停车收费单，纸已经湿透了。江叙白站在门廊下，右手插进夹克口袋里，指尖碰到那把黄铜钥匙的齿槽。钥匙是凉的，暗红色的痕迹在指腹的触感里清晰可辨。

他没低头看，只是握紧了一下，然后松开。口袋里的手机震了一下。

苏蔓的信息：“江老师，营业厅说异地通话记录要省公司调取，至少三个工作日。”

江叙白看完消息，把手机屏幕转向何沛。

“三天正好。”何沛发动车子，雨刷来回刮了两趟，“你明天下午去趟宏远物流的江湾分拣中心。刘建国在那儿干了三年，总有人还记得点什么。”

江叙白没上车。何沛也没等他回答。

桑塔纳的尾灯消失在雨幕里之后，江叙白在门廊下又站了一会儿。雨水沿着档案馆的排水管往下淌，铁皮管道里回荡的水声像有人在很远的地方敲一面闷锣。他掏出手机，给苏蔓回了条消息：“知道了。你帮我查一件事：苏皓在隆兴路老宅住的时候，有没有去守过快递代收点。”

发完消息，他把衣领翻起来，走下台阶，走进雨里。

口袋里，那把七号钥匙硌着掌心。老覃说的那句话还在他脑子里转：“也是个雨天来的。”
江叙白没直接回家。他在雨里走了二十分钟，拐进惠民路一条巷子，推开了“四季春”茶餐厅的玻璃门。

下午两点四十分，店里没有别的客人。角落卡座里坐着一个穿灰色polo衫的男人，面前摆着杯冻柠茶，杯壁上的水珠已经淌了一滩在桌面上。男人看见江叙白进来，抬手招了一下。

“江生。”他往前推了推对面的杯子，是杯热鸳鸯，还没动过。

江叙白坐下，脱了夹克搭在椅背上。椅子是那种老式火车座，人造革面裂了几道口子，露出里面发黄的海绵。他把右手放在桌面上，手套没摘。

“你电话里说的事，我查了。”灰衣男人叫周兆麟，江叙白在《南方都市报》做调查记者时期的线人，现在自己开了家商务调查公司，专门接保险公司和律所的外包活。“江湾码头那个案子，九八年八月立案，十月归档。归档人签名是当时的刑侦支队长，叫葛振国。”

周兆麟从公文包里抽出一个牛皮纸信封，推到杯子旁边。

“葛振国九九年三月调任省厅，零四年病退，一零年肺癌去世。他在任期间经办的重案一共七宗，结案率百分之百。”周兆麟顿了顿，“包括江湾码头沉箱案。”

江叙白拿起信封，没急着打开。

“结案结论是什么。”

“意外事故。码头扩建，施工方在七号泊位水下浇筑沉箱，五个工人下到箱体内作业。下午涨潮，水位上涨速度超出预测，沉箱底部进水，五个人没能及时撤离。”周兆麟喝了口冻柠茶，冰块在杯子里响了一下，“调查结论是施工方未按规范设置水位监测，负责人被吊销资质。死者家属各获赔七万八。”

“五个工人。”江叙白说。

“对，五个。名字、籍贯、年龄我写在纸上了。”

江叙白打开信封。里面是三张打印纸和一张彩色复印件。复印件是一张当年的《江湾晚报》，日期一九九八年八月二十一日。头版下方有一条新闻标题：《江湾码头施工事故致五人死亡，警方初步排除刑案可能》。

他把复印件放在一边，翻打印纸。第一页是五个死者的基本信息。三个四川人，一个湖南人，一个本省肇庆人。年龄最大四十七，最小十九。第二页是事故调查报告摘要，没有异常。第三页是周兆麟自己写的备注，字迹潦草：

“死者中有一人身份存疑。四川达州籍工人刘建国，当年登记的身份证号经公安网比对，对应户籍已于一九九六年注销。注销原因为‘迁出’。但达州那边没有接收记录。换句话说，九八年死在江湾码头沉箱里的那个刘建国，用的是一张已注销身份证。”

江叙白把这一页单独抽出来，放在最上面。

“苏皓的失踪案卷里提到，他失踪前去江湾码头找过一个叫刘建国的人。”他说，语气不是陈述，是确认。

周兆麟往后靠在椅背上。他看江叙白的眼神变了，不是惊讶，是一个做了二十年调查的人终于听到一个能对上号的线索时，那种压着兴奋的冷静。

“那就有意思了。”周兆麟说，“一九九八年八月，刘建国死在沉箱里。你的档案里说，苏皓失踪前在江湾码头找刘建国。但苏皓报失踪是十月。两个月的时间差。”

“如果刘建国八月就死了，苏皓十月去找的是谁。”江叙白把话接过去。

茶餐厅的空调出风口正对着他们的卡座，冷风把冻柠茶杯壁上的水珠吹得往下滑。江叙白把五份档案里那个叫刘建国的四川达州人的信息，和周兆麟带来的材料并排放在桌面上。姓名一致，籍贯一致，身份证号差了一位。档案里登记的和当年身份证注销记录里的，尾号一个3一个8。

“这种误差如果是登记时的笔误，”周兆麟说，“那写错的人，要么根本没见过刘建国的身份证原件，要么他不在乎这个人是谁。”

江叙白把材料收回信封。他站起来，从裤兜里摸出两张一百块压在杯子底下。

“帮我继续查刘建国。我要知道他九六年注销户籍之前在达州的社会关系，有没有兄弟姐妹，父母还在不在，有没有人认领过他的遗骸。”

“遗骸？”周兆麟皱了下眉，“尸体没被家属领走？”

“何沛那边帮我确认过了。五个死者，四个家属签了火化同意书。只有刘建国，尸体在殡仪馆停了四个月，无人认领，最后作为无名尸火化处理。”

周兆麟沉默了几秒。然后他点了下头。

江叙白推门出去的时候，外面的雨已经小了，变成一种细密到几乎看不清的雾雨。他沿着骑楼往东走，走过了三个路口才停下来。骑楼的廊柱上贴满了小广告，最上面一层是新的，底下几层被雨水泡烂又晒干的纸浆叠在一起，像某种地质岩层。

他背靠在一根廊柱上，掏出手机。苏蔓半小时前又发了一条消息：“江老师，快递代收点的事我明天去问。还有，今天下午有个姓聂的老人打电话到报社，说找您。他说您知道怎么找他。”

江叙白盯着这条消息看了五秒钟。聂守仁主动联系他。这不在他的预料之内。

他拨了守拙斋的座机号码。响了三声，通了。

“聂师傅，我是江叙白。”

电话那头安静了一会儿。然后聂守仁的声音慢慢传过来，沙哑，不急不缓。

“江先生，你上次问我的那件事。”

他停了一下。听筒里传来椅子挪动的声音，然后是关窗的动静。

“我这里还有一把钥匙。八号。今天下午，有人把它当进来了。”
