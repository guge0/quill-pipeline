# 第2章 创作者细纲

## 戏核
这一章的存在是为了 **把上一章获得的线索推到台面，用低密度对话场景完成一次情绪沉淀和信息筛选，同时让一个被忽视的线索浮出水面——宏远物流，以此制造新的叙事推力**。
读者读完这章，记住的东西应该是 **最后那通两分十四秒的电话，以及一个物流公司的名字，比清单上的所有数字都重**。

## 戏核如何承载(含技法)

这一章的核心技法选择是 **慢节奏特写 + 静物镜头**。全程节奏压住，不让任何情绪冲到表面。

开篇直接切进茶餐厅内景，用一组固定机位的中景——两人对坐，桌上叠着三十七页纸。这里不需要渲染气氛，气氛是雨声和茶餐厅暖光自动给的。用两个反打把苏蔓的状态交代清楚:她翻清单时手指压在页边，不翻动只看着某一页不动。这是表演的瞬间，告诉读者她一定看过很多遍了，现在只是陪江叙白再走一遍。

笔墨要砸在 **翻阅清单的过程** 上。这里需要一种近乎枯燥的如实呈现——江叙白翻页，苏蔓偶尔补充一句注释，间或沉默。不是每个条目都要写，但要用几处具体的号码和时长制造真实感。真正关键的是 **倒数第二条**:一个空号，苏蔓确认之前查过。然后翻到最后一页，倒数第一条——宏远物流，两分十四秒。这是戏核所在的瞬间，要用特写:江叙白的手指停在那一行，两秒钟的静止，然后他抬头看苏蔓。苏蔓说“他从来没提过物流公司”。这句话不能带情绪，要平静，平静比激动更有效。

回忆部分要用 **闪入式白描**，不能展开成回忆段落。苏蔓说“灰色连帽衫”“取个东西”时，用一个短镜头跳切到前景:茶杯上的水汽模糊了对面的脸，然后切回。这种用法克制，且不打断审讯般的一问一答节奏。

左肩旧伤在章中只出现一次——江叙白在苏蔓低头翻包时，有一个微不可查的左肩前旋动作，然后他停止了这个动作，继续翻页。读者如果能注意到，是给他们的小奖励；如果注意不到，也不影响戏核传达。

章末告别是一场 **留白对话**。没有“小心”“你也是”，只有“下周二档案馆门口”和“嗯”。然后江叙白站起来，苏蔓没动。镜头不是跟江叙白出门，而是停在茶餐厅外——苏蔓透过玻璃看江叙白在雨里站了一下，然后转身。这时候镜头推近玻璃上的雨痕，背景里有一把黑伞，停在行道树旁，伞沿压低，站了很久。这个钩子不能给暗示性镜头，伞就是一把伞，但停留时长异常，让读者自己感觉到不安。

## 笔墨分配

- **关键处**(开篇至全章约60%):从苏蔓出现在茶餐厅门口到清单翻完、宏远物流线索浮出水面。这四个段落(相见→咖啡→翻页过程→宏远物流)是戏核所在，要展开、要细。翻页过程要用具体的纸页沙沙声、翻到某页手指停住的瞬间、苏蔓偶尔的轻声注释来填充，不能空。

- **过渡处**(约25%):苏蔓回忆弟弟出门的片段、江叙白旧伤的暗示、告别对话。这些段落是氛围支撑，用一句话带过的动作或对话交代即可，不需要渲染情绪。

- **删减判断**:清单的详细内容不需要逐条展现，只需展示3-4个有代表性的号码和最后一页的宏远物流。雨的外景描写只出现在开场一句和章末一句，中间不重复。茶餐厅内部陈设、上菜过程全部略过。

## 与前后章的关系

- **承**:准确衔接第一章结尾——江叙白约苏蔓取清单，“你带，我请你喝东西”。本章开场直接验证了这个约定成立。第一章获得的“守拙斋”线索、当票、黄铜钥匙细节在本章对话中被苏蔓不知道/没见过/没听他说过——通过否定来加深线索的量。

- **启**:宏远物流是本章创造的新追踪路径，将在第三章中去调查。章末的伞可以有两个方向:一个方向是单纯的路人(误导)，另一个方向是盯梢者(伏笔)——这个判断交给下一章的创作者，本章不揭示。同时，本周二档案馆之约被确立，为第三章的场景转移做铺垫。

## 硬信息锚点块

- **时间**:周五下午
- **地点**:苏蔓说地址“上次来过了”——即江叙白住处楼下的茶餐厅
- **人物**:江叙白, 苏蔓
- **数字**:三十七页, 两分十四秒
- **约定**:通话清单——苏蔓如约送来
- **设定**:宏远物流, 灰色连帽衫, 左肩旧伤
- **设定**:江叙白左肩旧伤阴雨天隐隐作痛——已设定但未在前文出现，本章确证
