# 第2章 创作者细纲

## 戏核
这一章的存在是为了让江叙白和苏蔓从“信息求助关系”正式进入“共同调查关系”——通过清单这一具象物,把苏皓从抽象失踪者的标签还原成一个有具体生活轨迹的弟弟。读者读完这章,记住的东西应该是:苏皓失踪前最后联系的是宏远物流,他穿了件灰色连帽衫出门,说“去取个东西”——这三件事叠在一起,构成了一个清晰但危险的日常缺口。

## 戏核如何承载(含技法)

本章的戏核不在情节推进,而在**情绪蓄力和线索编织**。剧情简介的事件密度低,恰好是作者主动选择的节奏控制——上一章守拙斋的气氛太紧,本章需要一段“相对正常”的茶餐厅对话来让读者喘息,同时也让江叙白(和读者)借助苏蔓的口述,把苏皓这个角色从“第六名失踪者”补全为“会穿灰色连帽衫、会去取东西的年轻人”。

核心场景是苏蔓回忆弟弟失踪那晚的那段话。技法上可以用**慢节奏的近景切换**:茶餐厅的暖黄灯光下,清单纸页上的墨字,苏蔓的手握紧茶杯又松开,眼神从江叙白脸上移开落到窗外雨幕。这段回忆不应该被打断,让它自然流动,直到苏蔓说完“很快回来”,再切回江叙白的反应——他折起第三十七页。这个动作是本章的情绪锚点:他没有追问、没有安慰,而是用行动表明他相信这个细节值得追查。

章末的黑伞镜头是**长镜头式的固定机位**,从江叙白和苏蔓告别后,镜头停留在茶餐厅玻璃内侧,雨幕模糊了视线,拍摄黑伞在路灯下静止不动。时间长度要足够让观众/读者产生“这不是巧合”的判断。伞下人物不露脸、不进店,只构成氛围威胁——和上一章守拙斋的诡异形成互文,证明调查的危险已经尾随到了日常空间。

本章的情绪基调是**克制的温情与渐浓的寒意并行**:茶餐厅内的对话是温和的、有序的,场景甚至有些日常感;而窗外雨、旧伤隐痛、最后一页折角、黑伞滞留——这些细微的“不安”不断叠加,在章末形成一次无声的高潮。

## 笔墨分配

- **关键处(中段至章末,约占全章 55%)**:苏蔓回忆弟弟穿灰色连帽衫出门、“去取个东西”的段落;江叙白折起第三十七页清单的细节;章末黑伞的静置镜头。这三处是情感和悬念的核心,需要展开细节描写和氛围具象化。

- **过渡处(开篇及中段前半,约占全章 30%)**:清单交接的流程、两人坐下点单、整理清单过程中对已知线索的简要复述(如基站位置、警方结案情况)——这些是必要的信息校准,但不要在此处展开新推理或新情绪,笔墨简省,以对话推进为主,避免内心独白。

- **删减判断**:原剧情点 1 中“事件密度低”的判断本身不需要在正文中说明;情节简介第 4 点“左肩旧伤疼痛”不必展开描写或做人物回忆,只需在苏蔓说话时提及一句(如江叙白换了个坐姿、手不自觉地碰了一下左肩),保持它的“体感真实”而不是“符号提示”。

## 与前后章的关系

- **承**:承接上一章守拙斋记录仪丢失 64 秒、江叙白左肩旧伤在雨夜发作的生理暗示;同时接住了 hook-001(苏皓最后通话对象)的追问,把焦点从“基站位置”转向“宏远物流”这个具体名词。

- **启**:本章明确了两个后续方向——档案局调阅 A-113 旧案和实地碰宏远物流,为第 3 章的两线推进提供了动机依据;章末黑伞的出现则把守拙斋的危险延伸到了江叙白的日常追踪中,为下一章的风向转变做悬念储蓄。

## 硬信息锚点块

- **时间**:周五下午
- **地点**:江叙白住处楼下的茶餐厅
- **人物**:江叙白、苏蔓
- **数字**:三十七页、两分十四秒
- **约定**:苏蔓把苏皓的通话清单送来
- **设定**:宏远物流、灰色连帽衫、左肩旧伤
