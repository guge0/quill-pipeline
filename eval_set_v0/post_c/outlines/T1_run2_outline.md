# 第1章 创作者细纲

## 戏核
这一章的存在是为了用一次“看似平常的夜访”，把读者拽入一个“表面正常但处处不对劲”的世界。读者读完这章，记住的东西应该是 **“守拙斋里的人、那把钥匙、和那段消失的录像，都透着让人不安的秩序感”**——不是惊吓，是持续的、低沉的异样感。

## 戏核如何承载（含技法）

核心场景是江叙白在守拙斋内的八分钟。这里用**慢节奏**和**固定机位式的长镜头**处理：聂守仁几乎没动过位置，镜头（读者视线）却不断被牵引——从他脸上的神情滑到左手黑色手套，再滑到背后那扇带第二道门的卷帘门，再回到柜台上的黄铜钥匙。每一处细节都用**特写**刻一下，但不解释。聂守仁的台词要像“留白的水墨画”，每句话后半句都像被咽回去了。

另一个关键瞬间在章末：江叙白在台灯下翻过钥匙，发现第二行小字。这里用**慢推镜头的特写**——钥匙从全景推进到柄部，光从侧打，字迹从模糊到清晰，然后在读者来得及看清内容之前停住，切黑。不给答案，只给反应（握紧）。

何沛与江叙白在粥铺的对话用**快切**式处理，信息密度高但节奏轻快，与守拙斋内的“滞重感”形成反差。这个场景的作用本质上不是揭露信息，而是**通过何沛的口，把“六名失踪者”“四起结案干净得过分”“苏皓最后一个电话在回声巷”这些零散锚点串成一条脉**——让读者意识到：不是江叙白多疑，是这个案子本身有结构性的问题。

## 笔墨分配

- **关键处**（约占全章55%）：中段守拙斋内场景 + 章末钥匙柄小字发现。前者以慢节奏、留白和细节特写推进，后者用快切式的瞬间高光收束全文。
- **次要展开**（约占30%）：何沛与江叙白在粥铺碰面。笔墨用在何沛的表情、语气的谨慎与割裂感（一边说“四起结案干净得过分”，一边说“先别单独进当铺”），而非大量信息对白。信息点靠何沛的三句话带完，不展开。
- **过渡处**（约占15%）：开头雨中抵达回声巷、报刊亭老板的闲谈。只写雨、写脚步声、写语气冰冷的叙述，用氛围带过信息，不做氛围渲染之外的事。

## 删减判断

- **苏蔓通话清单的细节**，sub-md中何沛提了一句“周五下午送到”。这一句在正文中可以**一笔带过**，甚至只要何沛说“周五东西到”就够了——清单内容不是本章悬念的构成部分。
- **江叙白安装、调取微型记录仪的具体操作**，可删去技术细节，只写结论：“录像缺失四十一分钟”。读者不需要知道怎么调、怎么装的。
- **市档案馆三楼管理员老覃**，本章不出场，只如何沛一句提“我认调阅函，你认我”，算伏笔，不展开。

## 与前后章的关系

- **承**：接过上游的线索（苏皓最后通话在回声巷、六名失踪者名单），用一次夜访把“线索”转化成“经验”——江叙白亲眼见到了守拙斋的异样，读者也看见了。前文悬着的“为什么是回声巷”被落地为一个具体场景。
- **启**：章末钥匙柄第二行小字是给第2章留的钩子——第2章开头必须揭示小字内容，否则读者会失去对“这把钥匙为何重要”的感知。同时，何沛提到的“A-113旧卷”和“三天后档案馆碰头”为第2章定好了行动线。
