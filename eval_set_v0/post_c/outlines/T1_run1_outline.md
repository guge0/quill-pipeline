# 第1章 创作者细纲

## 戏核
这一章的存在是为了**建立“守拙斋”的规则与异常，同时制造第一个不可逆的悬念——录像缺失**。
读者读完这章，记住的东西应该是**那把07号钥匙和那四十一分钟的空白**。

## 戏核如何承载(含技法)

核心场景只有一个：守拙斋内的对峙与离店后的录像核查。这是本章的“定调时刻”——用慢节奏、长镜头式的静默铺陈来建立聂守仁的压迫感。柜台两端的距离感要清晰：江叙白站在店堂前半段，聂守仁在柜台后，两人之间隔着三米左右的“无人区”。摄像机视角停留在江叙白肩后，不切聂守仁近景，保持信息遮蔽——读者和江叙白一样，看不清对方表情细节，只能看到那只黑色手套始终平放在台面上。对话节奏要留白：关键台词之间插入三到五次静场，用雨声填空。

离店后的录像检查是章末重锤，用快切手法：记录仪回放、时间戳特写、江叙白拇指反复划拉进度条的动作三连切。发现缺失瞬间需一个沉默特写——他身体定住，雨水从额角滑到睫毛不眨。何沛碰面戏是过渡兼信息锚点，节奏平缓，用一碗粥的时间把何沛的态度、调阅函的安排、苏蔓的进度交代干净，不渲染情绪，不给任何画面化的细节（粥铺环境、表情特写一律省略）。章末钥匙小字是“收尾冷击”——光线来自台灯，钥匙在指间翻动的特写，笔尖停顿的两个节拍。

## 笔墨分配

- **关键处**(约占全章70%)：守拙斋内戏（从避雨进店到钥匙入袋）+ 出店后录像核查。这是戏核所在，要展开、要细、要建立节奏。尤其是聂守仁递钥匙和收钥匙两处动作，笔墨要重，让读者记住这个“被塞进外套口袋”的触感。

- **过渡处**(约占全章20%)：开篇雨夜抵达+报刊亭对话。必要的氛围铺垫和信息确认（亮灯传言），但不能抢戏，用短句和客观描写带过。何沛碰面戏前半段（寒暄、点单等）略写。

- **删减判断**：前文摘要中苏蔓“情绪焦虑但配合”的状态在本章不必着墨——她不出场，只需在何沛口中带一句“通话清单周五送到”。江叙白返家途中不写心理活动，留白比自言自语有力。报刊亭老板的闲谈中与主线无关的闲聊一律砍掉（如天气预报、周边趣闻），只保留“只有17号雨夜亮灯”这条信息。

## 与前后章的关系

- **承**：接住前文留下的两个开放钩子（苏皓通话对象、四起快速结案），本章通过何沛口述初步回应结案疑点，但未打开。江叙白对守拙斋的物理接近，是从“档案照片”到“实地接触”的突破。

- **启**：留下三个推进路径——07号钥匙（去守拙斋赎什么？）、“A-113沉箱案”（三天后档案馆）、“录像缺失四十一分钟”（技术与外部干预的线索）。钥匙小字是章末冷击，也是第2章开篇的接应点。

## 硬信息锚点块

- **时间**: 周四夜里十一点二十, 三天后上午十点, 周五下午
- **地点**: 回声巷巷口, 回声巷17号, 守拙斋, 江湾区粥铺, 市档案馆三楼
- **人物**: 江叙白, 聂守仁, 何沛, 苏蔓, 报刊亭老板
- **数字**: 六名失踪者, 三个月, 四十一分钟, 07号黄铜钥匙, A-113
- **约定**: 三天后上午十点在市档案馆三楼碰头（何沛带调阅函）, 在那之前江叙白不单独进当铺, 苏蔓周五下午把苏皓通话清单送到江叙白手上
- **设定**: 聂守仁左手戴着黑色手套, 守拙斋只在雨夜亮灯, 店堂尽头卷帘门内侧有第二道门挂着锁, 钥匙柄刻着编号07, 钥匙柄内侧刻着第二行小字（本章不揭示）
