# 第3章 创作者细纲

## 戏核
这一章的存在是为了让“系统内部有人”这条暗线从悬疑变成确证。读者读完这章，记住的东西应该是**档案馆里那张被水渍洇开的签名栏——何沛的权限都挡不住有人在系统里动手脚**。钥匙硌手心是情绪收尾，不是信息推进。

## 戏核如何承载（含技法）

章首用“静”来压场。开篇三句话：十点整、江叙白推门、何沛已经在窗边。两个人交换一个眼神，谁也不提当铺那三天的事——**默契本身就是信息**，告诉读者：这两人之间的信任不需要台词来填。调阅函交接用一个特写：何沛手指捏着纸角递过去，江叙白接的时候没看纸面。这0.5秒的动作就完成了“守约”的确认。

翻卷宗的段落用**慢节奏的细节序列**。三个人各自翻不同部分，镜头在三人之间平切：何沛看笔录签字页，江叙白翻证物清单，老覃站在柜台后面擦眼镜——这是一个固定机位的长镜头感，让读者感受到档案室的安静和纸页的触感。直到何沛的手指停在某一页边缘，空气突然收紧。这里用**快切**：何沛抬头→江叙白看他手指位置→老覃放下眼镜走过来。缺页被发现，不是通过台词宣布，而是通过三人视线在同一张纸上的聚拢来完成。

何沛那句“这卷宗本不该有人能在登记不全的情况下调走内页”，要用**低音量的对峙节奏**。他不看江叙白说这句话，而是对着卷宗说的——这意味着他第一反应不是对江叙白解释，而是自己在消化一个判断。老覃的迟疑和那句“也是个雨天来的”要留气口。章末单独成段，江叙白口袋里钥匙硌着手心，只写触感，不写心理活动——**留白让读者自己完成“这钥匙和缺页之间是否有关系”的推理**。

## 笔墨分配

- **关键处**（约占全章55%）：发现缺页的段落、何沛脸色变化后说出“这不应该”的段落、老覃说出“也是个雨天来的”之后的静场。这三个节点逐层递进，是戏核的承载区，需要精确的节奏控制和细节密度。
- **过渡处**（约占全章30%）：章首到达与碰头、调阅函查验与交付、翻卷宗的初始过程。这些是必要的建立情境，但不需要渲染，用动作和环境的冷描述即可。
- **删减判断**：三人翻卷宗时的具体内容展示——这是T2已经铺垫过的沉箱案背景，本章不需要再复述，只需要让读者知道“他们在看这个卷”即可。老覃验函的过程也不需要展开写，一笔带过。

## 与前后章的关系

- **承**：T2结尾留下的“江叙白守约三天没进当铺”在本章开篇用调阅函一笔带过完成呼应。T1里何沛“讲义气但讲程序”的双面性，在本章通过他亲手签调阅函来完成。
- **启**：本章为T4留下两个必须回答的问题：缺的第三页去哪了？上一个雨天来调卷的人是谁？同时，老覃那句“也是个雨天”叩响了守拙斋“只在雨夜亮灯”的设定，为两条线最终交汇埋下坐标。

- **地点**：市档案馆三楼
- **人物**：江叙白，何沛，老覃
- **数字**：A-113，第三页，07号
- **约定**：调阅函，不单独进当铺
- **设定**：1998年“江湾码头沉箱案”
