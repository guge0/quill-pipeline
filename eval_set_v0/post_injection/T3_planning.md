# 第 3 章 创作者细纲

## 戏核
这一章的存在是为了**让“档案被人动过手脚”这个事实从猜测变成可触可感的证据，同时把聂守仁的身份疑云和1998年的旧案做第一次实质性焊接**。
读者读完这章，记住的东西应该是**那卷缺了第三页的卷宗——有人比何沛权限更高、比档案馆流程更硬，在某个雨夜悄悄取走了关键的一页**。

## 戏核如何承载(含技法)

核心场景锁定在三人翻卷宗的段落。不要一上来就交底，要让读者先经过一段“正常”的查阅流程——老覃验函、取卷、灰尘扬起来、纸张脆响——建立起“这是一次常规操作”的错觉。用慢节奏推进：手指划过目录、日期、编号，光线透过窗户落在纸面，安静得只剩下翻页声。等到缺第三页被发现的那一刻，节奏突然卡住。写何沛的手指停在页码处，写他重新数了一遍页数，写他看卷宗脊线有没有拆过的痕迹——这一连串动作用近乎静默的特写处理，不给任何内心独白或台词。江叙白的反应放在第二个层面：他不说话，只是观察何沛的表情变化，同时用余光留意老覃。

借阅登记卡上的水渍是第二个重笔。这里用一次闪回式的信息嫁接：何沛认出签名栏位置时，切一个极短的回忆画面（他五年前在刑侦队时见过的某次调阅流程，作为对比）。但不展开，只是闪一帧，立刻切回当下。他用指腹摩挲水渍边缘，判断洇开时间——这是专业嗅觉的自然流露，不是刻意炫技。老覃迟疑的那句台词要拍成全景中的人物关系缩紧：他说完，三个人都静止了一瞬。

章末江叙白口袋里的钥匙硌着掌心，不用写他摸钥匙的动作，只写他感觉到那枚金属的存在——一个触觉上的提醒，把前两章的物证和这一章的档案疑云串起来。

## 笔墨分配

- **关键处**:中段“发现缺页+登记卡水渍+何沛的专业反应”这组连贯场景，约占全章55%。这是戏核所在，要细到能看见每一根手指的动作。章末老覃的台词及其之后的沉默，约占15%，作为情绪落点。
- **过渡处**:开篇江叙白进档案馆、与何沛简短碰面约占20%，写清时间地点和两人状态的细微差别即可，不必展开寒暄或往事。老覃验函取卷过程约占10%，只写必要的流程动作，不写心理活动。
- **删减判断**:sub-md中“江叙白这三天也守了约没单独进当铺”——这条在三章内不需要展开，一笔带过即可。它在sub-md里的作用是解释人物履约状态，但这一章的核心是卷宗，不是江叙白是否听话。如果正文里写他对何沛说“我这三天没去过当铺”，反而会削弱档案馆场景的紧张度。

## 与前后章的关系

- **承**:CH2中江叙白和何沛的约定(调阅函+不进当铺)在这一章开头即兑现，两人关系从试探进入有限信任。老覃的“雨天”台词直接呼应CH1守拙斋“雨夜亮灯”的设定，暗示聂守仁不是随机出现在雨夜，而是雨天有档案调阅的某种关联。
- **启**:拉开一条新的行动线——缺页里写了什么、是谁取走的、和聂守仁什么关系。这章给CH4留下的伏笔是：何沛会回去查借阅系统的操作日志，江叙白则会想办法确认老覃口中的“上一个雨天来的人”是否就是聂守仁。

- **地点**:市档案馆三楼
- **人物**:江叙白；何沛；老覃
- **数字**:档案编号A-113；第四起结案(对应“六名失踪者，结案四起”设定)
