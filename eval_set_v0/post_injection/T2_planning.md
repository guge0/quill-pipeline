# 第2章 创作者细纲

## 戏核
这一章的存在是为了**在低事件密度中建立苏蔓这个“委托人”的真实感，同时让“宏远物流”从一个号码变成第一个值得深究的指向**。
读者读完这章，记住的东西应该是**茶餐厅里对坐的两个人各怀心事，以及窗外那把不进来的黑伞——有人在看他们，但他们还不知道是谁**。

## 戏核如何承载(含技法)

本章的核心不是信息量，是**氛围和关系**。三十七页清单是道具，不是内容——读者不需要逐条看通话记录，需要看到的是**江叙白在翻页时的反应，苏蔓在回忆时的手势，以及这两个人之间的沉默和弹性的张力**。

戏核落在两个地方。第一个是**苏蔓回忆弟弟出门的那段话**——这里用特写：她把咖啡杯转了两圈才开口，说的内容是“灰色连帽衫”“取个东西，很快回来”，但镜头应该推在她说话时的嘴唇和指尖，而不是弟弟的背影。因为她说的内容其实在sub-md里已经给了，不值钱；值钱的是**她说这话时的节奏——停顿、加速、再停顿——一个护士职业的条理和姐姐身份的情绪拉扯**。

第二个戏核是章末的**黑伞**。这个钩子要用长镜头处理：雨幕、玻璃、伞沿滴水、看不见人脸。茶餐厅内的暖光和伞下的冷调形成对照。伞不动，人不进，但读者会替江叙白紧张——因为江叙白背对窗户，没看见。**这场戏的核心功能是让读者意识到：他们以为自己在暗处调查，其实已经在被人注视**。

中间的清单核查、物流号码的引出、江叙白折角这三个动作是过渡，但折角动作本身可以用慢动作拉一下——手指压在第三十七页的折痕上，配合左肩隐痛（肩膀先痛，手才去折页，不是反过来）。这暗示他的身体比意识更早察觉到这张清单里的危险。

## 笔墨分配

- **关键处**：苏蔓回忆段落（约全章35%）+ 章末黑伞长镜头（约20%）。回忆段落要用演员式的表情和沉默的间隙来撑，而不是台词的信息量。黑伞部分要写够空间感——街道、雨声、店内灯光打在伞面上的角度。
- **过渡处**：茶餐厅点单、清单交接、号码展示、告别走人。这些加起来约40%，保持节奏不拖，但不能抢戏。清单细节用快切（几秒钟翻几页的感受），不要一页页细说。
- **删减判断**：sub-md里“三十七页”这个数字保留，但具体清单内容不需要展示给读者看——观众不需要看到每一页，只需要看到江叙白的眼神在某页停了一下。雨再下起来这个动作可以一笔带过，不需要重复描写氛围。

## 与前后章的关系

- **承**：第1章结束在苏蔓打来电话约“通话清单拿到了”这个点上，第2章直接接上茶餐厅的对坐。江叙白左肩旧伤的提示在第1章应该已经铺垫过雨天的关联，这里第二次出现，建立“阴雨天=身体信号”的规律。
- **启**：第3章是档案馆之约（沉箱案）。第2章章末江叙白的决定（去档案馆后碰宏远物流）是叙事指令——给第3章一个明确的任务链。同时黑伞这个观察者为后续建立“有人在监测江叙白的调查进度”创造条件，但这个揭示要在T3之后才用得上。

## 硬信息锚点块

- **时间**: 周五下午
- **地点**: 江叙白住处楼下的茶餐厅
- **人物**: 江叙白、苏蔓
- **数字**: 三十七页、两分十四秒、第三十七页
- **约定**: 档案馆之约后再去碰宏远物流
- **设定**: 江叙白左肩旧伤阴雨天隐痛；苏蔓在医院做护士，夜班多
