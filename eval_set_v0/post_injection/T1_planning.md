# 第1章 创作者细纲

## 戏核
这一章的存在是为了 **用一场克制而精确的雨夜闯入，在读者心里刻下“守拙斋不正常”的直觉，但所有证据都只指向“似乎正常”的边缘——让怀疑成为这章的真正主角**。
读者读完这章，记住的东西应该是 **那把钥匙和那四十一分钟录像缺失带来的不安感，以及一个人进了一家店、时间却被抹掉的诡异**。不是案情进展，是世界观的确立：这个世界里，有些东西在现实表面之下运作，但作者不会明说。

## 戏核如何承载（含技法）

**核心承载段是守拙斋内部的12分钟**。江叙白推门进店开始，使用长镜头式的跟随感——镜头先给雨夜的背景噪音（雨水砸在雨棚上的闷响、远处汽车驶过湿路面的声音），然后切到店内的静。这个静是反常的：没有收音机、没有挂钟滴答、旧货堆叠但落灰静止得像博物馆展品。聂守仁坐在柜台后，视角应当让读者先看见他完好的右手放在账本上，然后才扫到柜台下阴影里那只戴黑色手套的左手——慢节奏推进，让所有细节都有“被主动观察”的质感。

对话应当保持不对称的信息密度：聂守仁的话“先生是避雨，还是避别的什么”像是知道了什么，但他又主动岔开话题聊天气；他摆出钥匙时是一个强特写——放在柜台台面上，这个动作需要慢镜头式的拆解：手的动作、钥匙落在木头上的声音（比金属应该有的声音更闷）、江叙白没有立刻接这个动作。整个段落的核心技法是用**信息密度差**：聂守仁给的每句话、每个动作都像有第二层含义，但所有词句本身又构不成任何指控。读者会像江叙白一样，觉得“不对劲”，但说不清哪里不对劲。

出店后的录像缺失段是全章情绪转换的关节。技法上是**快速剪辑**：江叙白在巷尾冷静地调取记录仪、倒带、比对时间戳、确认缺失——用动作和数字（23:11-23:52，整整41分钟）代替心理描写。然后切一个他的微表情（皱眉，手指在记录仪外壳上敲了三下），切回他收设备走人。这一小段不用任何内心独白，用**节奏由慢到快再到骤停**来制造不安。

粥铺碰面这段应当压低节奏，何沛和江叙白的对话是对前段的消化与验证。何沛说“四起结案材料干净得过分”时，主镜头应该给何沛搅动粥碗的手——这个动作泄露出他的紧张或者说有所保留，因为何沛的性格是谨慎讲程序的，他不会无理由地额外关注一个案子。这个场景是整个第一章的信息锚点：把个人观察（录像缺失）与系统数据（结案异常）并置，形成读者的信息确认闭环。

## 笔墨分配

- **关键处**：守拙斋店内段落（从推门雨棚下的停顿到出店后录像缺失前）约占全章40%。这是戏核所在，要展开所有感官细节（气味、温度、光线、声音），对话须精炼但有重量，动作节奏要慢、要实。章末钥匙的第二行小字揭晓段落约占10%，作为钩子要写得克制——一个动作，一句叙述，停在握紧的瞬间。
- **过渡处**：何沛粥铺段约占30%，在“信息确认”意义上重要，但不必用大量篇幅渲染氛围，笔墨应放在信息交换的节奏和两个人物的微小异常（何沛搅粥、江叙白点烟等）上。报刊亭对话段约占15%，用于交代“守拙斋只在雨夜亮灯”的设定并从外部视角确认异常，但不必深入，点到即止。
- **删减判断**：sub-md中“江叙白在巷尾调取记录仪发现缺失”这部分，不必写他如何安装、如何操作设备的细节，一笔带过“调取-发现-确认”即可。何沛提到“苏蔓那边周五下午会把清单送来”这句话在第一章节奏上可以一笔带过，不必展开描写，因为它在本章不承担情绪或信息锚点功能，只是为下章埋线。

## 与前后章的关系

- **承**：第一章是开篇，没有前文承接。但需要建立两个前置状态：一是江叙白已经在追踪苏皓失踪案（通过他进巷时走路有目标的姿态、他对当铺的已有认知暗示），二是这条线是“第9天”——他已经查了一段时间，但这是第一次触碰到真正异常的东西。
- **启**：留给第二章的硬钩子有三个：1）钥匙内侧第二行小字（需要设计一个符合世界观的具体内容，例如年份、人名或地点缩写）；2）三天后档案馆A-113卷宗的约定；3）“守拙斋内部录像缺失41分钟”这个物理层面的异常，意味着第一章建立了世界观规则（有些东西可以被抹掉），第二章需要在此基础上继续推进。何沛这个角色在这里建立了一个“愿意帮忙但有自己的边界”的基调，为后续可能的冲突埋伏笔。

## 硬信息锚点块

- **时间**：周四夜里十一点二十、三天后上午十点、周五下午
- **地点**：回声巷巷口、回声巷17号（守拙斋）、市档案馆三楼、江湾区粥铺
- **人物**：江叙白、聂守仁、何沛、报刊亭老板、苏蔓（被提及）
- **数字**：十一点二十、六名失踪者、四起结案材料、四十一分钟、编号07、A-113
- **约定**：三天后上午十点市档案馆三楼碰头、何沛带调阅函来、江叙白在那之前不单独进当铺、苏蔓周五下午送通话清单
- **设定**：守拙斋只在雨夜亮灯营业、聂守仁左手常年戴黑色手套、江叙白五年前从市局刑侦支队离职、档案编号A-113对应1998年“江湾码头沉箱案”
